# ❌ エラー解決ガイド: `[Errno 2] No such file or directory`

## 🔍 このエラーの意味

`[Errno 2] No such file or directory` は、**ファイルやフォルダが見つからない**というエラーです。

## 🎯 よくある原因と解決方法

### 原因1: `slack` フォルダに移動していない

**エラーの例**：
```
[Errno 2] No such file or directory: 'config.py'
```

**解決方法**：

1. **コマンドプロンプト（cmd）を開く**

2. **`slack` フォルダに移動する**：
   ```
   cd "C:\Users\hanhj\OneDrive\デスクトップ\前田のSAMURAI\AI\テスト\slack"
   ```
   ※ パスが違う場合は、実際のパスに変更してください

3. **現在のフォルダを確認する**：
   ```
   dir
   ```
   以下のファイルが表示されれば OK：
   - `slack_send_message.py`
   - `test_connection.py`
   - `config.py`
   - `requirements.txt`

4. **それからコマンドを実行する**：
   ```
   python test_connection.py
   ```

---

### 原因2: ファイル名のタイプミス

**エラーの例**：
```
[Errno 2] No such file or directory: 'test_conection.py'
```
（`test_connection.py` を `test_conection.py` と間違えた場合）

**解決方法**：
- ファイル名を正しく入力しているか確認
- エクスプローラーでファイル名を確認してから、正確にコピー&ペースト

---

### 原因3: ファイルが存在しない

**解決方法**：

1. **エクスプローラーで `slack` フォルダを開く**

2. **以下のファイルがあるか確認**：
   - ✅ `slack_send_message.py`
   - ✅ `test_connection.py`
   - ✅ `config.py`
   - ✅ `requirements.txt`

3. **ファイルがない場合**：
   - ファイルが削除されていないか確認
   - 別の場所にコピーされていないか確認

## 🔧 トラブルシューティング

### Q: パスがわからない

**A**: エクスプローラーで `slack` フォルダを開いて、アドレスバーをクリックすると、パスが表示されます。
それをコピーして、`cd "パス"` として使えます。

### Q: 日本語のパスでエラーが出る

**A**: パスを `"` （ダブルクォート）で囲んでください：
```
cd "C:\Users\hanhj\OneDrive\デスクトップ\前田のSAMURAI\AI\テスト\slack"
```

### Q: `cd` コマンドが動かない

**A**: 
1. パスが正しいか確認
2. フォルダ名にスペースがある場合は、必ず `"` で囲む
3. エクスプローラーでフォルダを開いて、アドレスバーからパスをコピーする

### Q: `python` コマンドが動かない

**A**: 
1. Python がインストールされているか確認：
   ```
   python --version
   ```
2. エラーが出る場合は、`py` を試す：
   ```
   py test_connection.py
   ```

## 🆘 まだ解決しない場合

1. **エラーメッセージ全体をコピー**して保存
2. **どのコマンドを実行したか**を記録
3. **現在のフォルダ**を確認（`cd` コマンドで確認）
4. **`dir` コマンドの結果**を確認

これらの情報があれば、より具体的な解決方法を提案できます。
